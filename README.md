# -Convergence-Active-Push-System
一个帮助构建主动信息流，提高信息获取目的性的轻量web应用|A lightweight web application that helps build an active information flow and enhances the purposefulness of information acquisition.
# Web Content Pusher - 快速入门指南

---

## 🎯 5分钟完成首个核心操作

### 前提条件

- 操作系统：
	- Windows 10、11 
	- Linux（已测试ubuntu24.0.4、CentOS10、Fedora43、manjaro-kde26.0.4）
- 有效的网络连接
- 解压压缩包（windows不推荐放在c盘，linux放主文件夹）

---

### 步骤一：启动应用

#### Windows 用户
```
双击运行 install_and_run_windows.bat
```

#### Linux 用户
```bash
在解压后的文件夹内右键打开终端
sudo chmod +x install_and_run_linux.sh
输入密码
./install_and_run_linux.sh
```

> **注意**：
> Linux：首次安装需要输入 sudo 密码（脚本会自动打开弹窗），随后脚本会自动完成环境、依赖和配置。如果不慎在运行过程中关闭，删除venv文件夹从步骤一重新开始即可。
> 
> Windows：如果在c盘解压压缩包，有些电脑需要使用管理员权限启动脚本。

安装成功后会自动打开浏览器，访问 **`http://localhost:5000`**。

> **注意**：
> 这时windows系统会出现是否允许**python服务器网络权限**的弹窗，点击允许。
> 
> 如果偶然没有自动打开浏览器，按住ctrl键点击终端中的`http://localhost:5000`链接即可。

---

### 步骤二：添加第一个主题

1. 点击左侧边栏的 **"+ 添加"** 按钮
2. 在弹出的对话框中输入主题关键词，例如 **"人工智能"**
3. 选择数据源（选中简书、CSDN、B站）
4. 点击 **"确认添加"**

---

### 步骤三：开始爬取内容

1. 在左侧边栏点击刚创建的主题
2. 点击右上角的 **"抓取"** 按钮
3. 等待爬取进度完成（进度条会显示当前状态）
4. 爬取完成后，内容会自动显示在右侧内容区域

---

### 步骤四：浏览和筛选内容

1. **查看内容**：点击任意文章卡片可查看详情
2. **来源筛选**：点击主题名下方的来源标签（如"知乎"、"小红书"）筛选特定来源的内容
3. **收藏文章**：点击文章卡片上的 ⭐ 图标收藏文章
4. **标记已读**：点击文章卡片标记为已读
5. **主页信息流**：点击主页，即可看到所有内容的随机推送

## 📱 界面导航

- **主页**：返回主页视图

## 🚀 常见操作

### 如何添加新的数据源？

1. 点击顶部导航的 **"管理源"**
2. 点击 **"+ 添加站点"**
3. 填写站点名称、URL模板、选择器等信息
4. 点击 **"保存"**

### 如何查看爬取进度？

点击抓取按钮后，页面顶部会显示进度条和状态信息。

### 如何收藏文章？

点击文章卡片右上角的 ⭐ 图标，再次点击取消收藏。

## ❓ 常见问题

### Q: 爬取失败怎么办？

A: 检查网络连接，确保目标网站可访问。部分网站需要登录，可在 `cookies/` 目录添加 Cookie 文件。

### Q: 高级功能的教程？

A: 点击右上角的使用指南即可看到完整的使用教程。

### Q: 如何修改主题关键词？

A: 点击主题的 **"配置"** 按钮，修改关键词后保存。
